                                    CSS - Cascading Style Sheets


Css describe how HTML elements/tag are to be displayed on screen, paper or in other media.

CSS saves a lot of work. It can control the layout of multiple web pages all at once.

syntax:

{
	color:"red";
	background-color:"yellow";
}

External stylesheets are stored in CSS files

3 Types of CSS

1) Inline CSS //inline css apply on particular element <p style=property:value>
2) Internal Css
3) External Css 


1) Inline CSS

For one line -- <p style=property:value> ex. <h1 style="color:red;background-color:yellow;text-align:center">Hi,I am HTML
